# DSM Books - Referências Bibliográficas

Este projeto foi desenvolvido como parte da **Aula 06 - React Context** da disciplina de Laboratório de Desenvolvimento Web do curso de **Desenvolvimento de Software Multiplataforma** da Fatec. 

## Funcionalidades

* **Listagem de Livros:** Exibição de todos os livros disponibilizados no arquivo `books.json`.
* **Filtros Combinados:** Possibilidade de filtrar as referências de forma simultânea por **Disciplina** e por **Semestre**.
* **Navegação SPA:** Roteamento entre a página inicial e a página de filtros utilizando o React Router.
* **Tipagem Estática:** Desenvolvimento seguro e previsível com as interfaces do TypeScript.

## Tecnologias Utilizadas

* **React**
* **TypeScript**
* **Context API**
* **Material UI (MUI)**
* **React Router Dom**
* **Axios**
